# DexGen Notes

The supported runtime path for this repo is:

- `DexGen_Final_Colab.ipynb` for Google Colab.
- `DexGenApp/app.py` or the packaged app for the local macOS client.
- `dexgen/shared/prompt_templates.md` for prompt reference material.

The legacy `/v1` remote server and orchestrator stack has been removed to avoid drift with the current launcher/client flow.
